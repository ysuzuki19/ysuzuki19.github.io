ubuntuのaptのkept backの解消

ubuntu
linux
apt
fix

2022/11/06

はじめに

Ubuntu のapt upgradeがうまく行かず、タイトルのメッセージが出力される。

その修正方法をメモとして残す。

発生事象

背景

Ubuntu の apt でパッケージを更新しようとした。

まず、apt updateで更新されているパッケージの存在が確認された。

$ sudo apt update
...
...
...
Reading package lists... Done
Building dependency tree... Done
Reading state information... Done
2 packages can be upgraded. Run 'apt list --upgradable' to see them.

更新対象となるパッケージ一覧は以下。

$ apt list --upgradable
Listing... Done
gnome-shell-common/jammy-updates,jammy-updates 42.5-0ubuntu1 all [upgradable from: 42.4-0ubuntu0.22.04.1]
gnome-shell/jammy-updates 42.5-0ubuntu1 amd64 [upgradable from: 42.4-0ubuntu0.22.04.1]
エラー文

実際にアップグレードをしようとしたところ、以下のようになった。

$ sudo apt upgrade
Reading package lists... Done
Building dependency tree... Done
Reading state information... Done
Calculating upgrade... Done
Try Ubuntu Pro beta with a free personal subscription on up to 5 machines.
Learn more at https://ubuntu.com/pro
The following packages have been kept back:
  gnome-shell gnome-shell-common
0 upgraded, 0 newly installed, 0 to remove and 2 not upgraded.

解消方法

以下のコマンドで、kept backが解消された。

$ sudo apt install --only-upgrade gnome-shell

実際、apt updateとapt list --upgradableで問題の解消が確認された。
