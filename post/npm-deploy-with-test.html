npmscriptsでデプロイする前にテスト通過を必須にする

npmdeploytest

June 14, 2021

はじめに

最近，テストを書く重要性を理解しつつある．

今回は，npm scripts のコマンドでデプロイする前にテストの通過を義務付ける設定をしてみた．

設定

package.jsonを書き換えるだけで簡単に設定ができる．

ついでに，ビルドも忘れないようにしてみた．

package.json
"scripts": {
    "test": "テストコマンド", #ex) jest --env=jsdom --verbose
    "build": "ビルドコマンド", #ex) next build
    "predeploy": "npm test && npm run build",
    "deploy": "デプロイコマンド" #ex) firebase deploy
}

おわりに

もっともっと安全にデプロイしたい．．．
