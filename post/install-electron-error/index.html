Electronインストールエラー対処

February 18, 2021

Electronを使いたいなと思い，インストールしたが，謎のエラーに悩まされた． その対処法を残す．

TL;DR

自分の環境の場合，以下のコマンドでインストールが無事完了した．

（グローバルのnpmを最新版に更新するので，実行は自己責任で！）

$ npm remove electron                               # ローカルのelectronを削除

$ sudo npm install npm@latest -g                    # npmの更新

$ sudo npm remove electron -g                       # 古いelectronを削除
$ sudo npm install electron -g                      # electronをグローバルインストール

$ electron                                          # => 成功

対処の流れ

結果的にエラー解消に直結したと思われる点は，本ページ上部にまとめたので， ここでは対処の流れを記しておく．

$ npm remove electron                               # ローカルのelectronを削除
$ npm install electron -g
$ sudo npm install electron -g                      # electronをグローバルインストール
$ electron                                          # => エラー

$ sudo npm install npm@latest -g                    # npmの更新

$ sudo npm install electron -g                      # => エラー
$ electron                                          # => エラー
$ 
$ sudo npm remove electron -g                       # 古いelectronを削除
$ sudo npm install electron -g                      # electronをグローバルインストール
$ 
$ electron                                          # => 成功
