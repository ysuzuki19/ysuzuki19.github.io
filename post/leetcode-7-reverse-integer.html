Leetcode 7 Reverse Integer

leetcodec++

May 25, 2020

はじめに

整数xの桁全てを反転した数値を求める問題

int型の範囲内に収まらない場合は0を返す。

<例>
123 -> 321
-123 -> -321

コード全体

int reverse(int x) {
  long xRev = 0;
  while(true){
    xRev += x%10;
    x /= 10;
    if( x == 0 ) break;
    else xRev *= 10;
  }
  if(xRev < INT_MIN || xRev > INT_MAX) return 0;
  return xRev;
}

解説
