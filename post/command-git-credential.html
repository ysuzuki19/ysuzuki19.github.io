UbuntuでgitのID/passwordを入力しなくて済む方法

machomebrewgit

December 03, 2020

Mac版gitはbrewでインストールすると、ID/passwordをキャッシュしてくれるが、 Ubuntuでは標準ではキャッシュしてくれない。

自宅の自分用デスクトップを使う限りは、ちょくちょく入力するのは面倒なので、キャッシュするように設定した。

コマンド

デフォルトでは15分間の保存とのこと。

$ git config --global credential.helper cache

キャッシュ時間を指定する

以下のように、指定することができる。 時間の単位は秒であり、タイムアウトを一日とする場合は、以下のようにする。

$ git config --global credential.helper 'cache --timeout=86400'

おわりに

他にも、SSHキーを用いる方法がある。
