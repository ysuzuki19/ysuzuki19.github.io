Ubuntu18.04のログインループへの対処

Apr 17, 2019
2 min read

#　最初に ubuntuのログインループへの対処は調べるといろんな方が紹介されいましたが、Ubuntu18.04LTSでの例は少なかったので今回解決した方法を紹介しておきます。

環境

Windows10とのデュアルブート Ubuntu 18.04 LTS

HP製PC Intel® Core™ i7-4510U CPU @ 2.00GHz NVIDIA Corporation GM108M [GeForce 840M]

問題

ユーザでログインする際、パスワードを入力すると一瞬固まった後暗転して再びユーザ選択画面へ戻る。

何度かログインを試しても同じ症状。

再起動して試しても同じ症状。

対処法

1.仮想コンソールを開く
2.unity-tweak-toolを再インストール
3.再起動

という手順で解決しました。詳しい手順は以下で説明していきます。

仮想コンソールを開く

ubuntuログイン画面でCtrol+Alt+F1で仮想コンソールが表示されます。

※ユーザ選択画面ではなくログイン画面

仮想コンソールを閉じたい場合は、Alt+F7もしくは

$ exit

とコマンドを入力するとログイン画面に戻れます。

unity-tweak-toolを再インストール

以下のコマンドを実行します。

$ sudo apt install unity-tweak-tool
$ unity-tweak-tool --reset-unity

途中、エラーが出たので以下のコマンドも実行しましたが、こちらは関係ないかもしれません。

$ rm var/lib/dpkg/lock
$ sudo apt --fix-broken install

ログインの確認

仮想コンソールを終了してログインができるかどうか確認。ログインできた場合は再起動してログインできるかどうか確認しましょう。 私の場合はここで解決していました。

上記の手順で解決しなかったら・・・

nVidiaドライバの方に問題があるかもしれません。そちらの方法を試してみてください。
